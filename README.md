PowerShotは、Unity C#で作成した3Dシューティングゲームです。
プレイヤーはボールを発射し、障害物を利用しながらターゲットを狙い、スコアを競います。
Raycastを用いた照準アシストや、WallManagerによる動的なステージ生成が特徴です。

Unityは、Unity Technologiesの商標です。

## ゲームの起動方法

圧縮ファイルを展開し、「PowerShot.exe」ファイルをダブルクリックしてください。

## アンインストール方法

展開したファイルをすべて削除してください。レジストリは使用していません。

## 工夫点・改善点
【工夫点・こだわった点】
- 視覚的な照準アシスト:
  `LineRenderer`と`Raycast`を組み合わせ、マウス位置に応じた弾道の予測線を表示し、プレイヤーが狙いやすい工夫をしました。
- 動的なステージ生成:
  `WallManager`により、1ショットごとに障害物の位置や角度をランダムに生成するようにしました。単調なゲームプレイにならず、リプレイ性を高めました。

【改善点・追加したい機能】
- ギミックの多様化（破壊可能な壁、減速する壁など）
- ゲームプレイの難易度選択の追加
- モバイル対応（タップ操作への最適化）
- ハイスコアのオンラインランキング機能
